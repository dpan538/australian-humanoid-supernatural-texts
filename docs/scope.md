# Project Scope

This project studies public textual representations of humanoid or humanoid-adjacent supernatural beings in Australian public texts. The geographic scope is Australia only.

The project does not build a generic monster database. It focuses on how public texts name, mediate, classify, popularise, and search for figures such as Yowie/Yahoo/Hairy Man clusters and selected broader humanoid or spirit-being clusters.

## Source Scope

Use only public, published, openly discoverable sources. Eligible source classes include public newspaper and magazine texts, public catalogue metadata, public heritage pages, modern public web pages, Google Trends, and Wikimedia pageviews.

Do not target secret/sacred, restricted, unpublished, community-controlled, private, or non-public Indigenous cultural materials.

## Time Architecture

- Primary semantic corpus range: 1842-present.
- Retrospective Trove backsearch window: 1803-1841.
- Modern Yowie subseries: 1970s-present.
- Google Trends attention series: 2004-present.
- Wikimedia pageviews attention series: July 2015-present.

## Inclusion Logic

Core V1 terms are seeded as `include_v1`. Candidate terms requiring human confirmation are seeded as `validate_before_include` and are not generated as normal core collection queries. Excluded and control terms are seeded as `exclude_core` or `control_only`.

Controls such as `Bunyip` and `Drop bear` are included only for comparison. Non-humanoid or culturally central figures such as `Rainbow Serpent`, `Ngatyi`, and `Bula` are excluded from the core corpus.
