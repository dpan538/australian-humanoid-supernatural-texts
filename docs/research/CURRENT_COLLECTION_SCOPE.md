# Current Collection Scope

This file supersedes the archived V1 narrowing reports as an engineering guide.
Those reports remain in `docs/research/archive/` as historical audit material,
but they no longer define collection gates.

## Working Position

The archive is a typed public-text corpus of Australian supernatural or anomalous
humanoid narratives, apparitions, spirit-person narratives, giant/ogre
narratives, local legends, encounter accounts, and materially distinct public
retellings.

The project should not be reduced to a Yowie archive. Yowie remains an important
semantic cluster, but current collection actively prioritizes non-Yowie source
labels and under-covered states/territories.

## Current Targets

- Public display corpus: 2500-3000 records.
- Strict map layer: 1500 records with verified coordinate-bearing locations.
- Dashboard and density views may include public display records with broad or
  pending geography.
- Map view includes strict coordinate-bearing records only.

## Priority Collection Axes

- Low-coverage states/territories: WA, NT, SA, TAS, VIC.
- Under-represented source labels: Mimih/Mimi, Garkain, Mokoi, Mamu,
  Pangkarlangu, Puttikan, Tjangara, Yaroma, Yara-ma-yha-who, Quinkan/Quinkin,
  Wandjina/Wanjina, Nargun, Hairy Man, Australian gorilla.
- Narrative types beyond encounter reports: apparition_account, ghost_legend,
  local_legend, traditional_narrative, spirit_person_narrative,
  giant_or_ogre_narrative, descriptive_belief_record, retelling_or_adaptation.

## Quality Rule

A record may enter dashboard/density when it has a stable public source, a
source label or narrative basis, an Australian relation, a neutral evidence
summary, publicness and ethics status, and a location role. It may enter the
map only when the location is a verified event, setting, legend-associated
place, or cultural-association place with coordinates.

Public institutional pages and public catalogue descriptions may support
records when the description itself is the public text being studied. They must
not be treated as permission to reproduce restricted cultural knowledge.
