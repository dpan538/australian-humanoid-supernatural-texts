# Collection 500 Plan

The target is 500 net-new accepted public source items after deduplication and inclusion gates.

The target is not 500 narrative units.

Do not count unresolved leads, metadata-only catalogue pointers, search-result snippets, exact duplicate URLs, controls, exclusions, inaccessible pages without sufficient evidence, AI-generated summaries, or unsourced listicles.

Diversification constraints:

- at least 450 accepted new source items should be non-AYR;
- no more than 50 should be newly accepted AYR items;
- at least 250 should be original, near-primary, public institutional, community-controlled public, or stable archival source items;
- at least 200 should be pre-1970 where safely obtainable;
- no single private paranormal aggregator should contribute more than 10 percent.

Collection proceeds in batches of no more than 50 accepted items. After each batch, run dedupe, schema validation, ethics checks, source-diversity audit, narrative-type coverage, exports, and a checkpoint.

When no Trove API key is available, Trove collector output remains a reproducible lead queue until article-level public content is verified.
