# Scope V2

Project name:

Australian Public Text Archive of Supernatural Humanoid Narratives and Encounters

This project is a typed, provenance-aware, ethically constrained archive of public Australian texts about supernatural or anomalous humanoid encounters, apparitions, rumours, legends, traditional narratives, and later retellings.

The governing question is:

How have supernatural or anomalous humanoid figures been encountered, narrated, localised, named, transmitted, and reinterpreted in Australian public texts?

The project documents public narratives and public records. It does not verify supernatural claims, treat labels as biological species, map habitats, claim comprehensive coverage of Australian traditions, or act as an authoritative Indigenous knowledge repository.

V2 adopts `narrative_unit` as the top-level research object. An encounter event is an optional subtype, not the universal model.
