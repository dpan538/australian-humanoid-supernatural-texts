# Ethics

This repository is designed for public-text research only. It must not be used to collect restricted, secret/sacred, unpublished, community-controlled, private, or non-public Indigenous cultural materials.

The project should not be framed as a monster database. The research question concerns public representation, semantic change, discourse, mediation, and attention signals. It does not treat all figures as the same kind of entity, and it does not collapse Indigenous cultural knowledge into cryptid categories.

## Coding Principles

Every record should preserve distinctions among:

- Source voice, such as Aboriginal author, settler journalist, museum, lexicographer, tourist copy, entertainment, scholar, or unknown.
- Genre, such as newspaper report, dictionary entry, museum page, heritage page, catalogue metadata, academic text, or web article.
- Publicness, such as open article, public metadata, open-access publication, public heritage page, or restricted excluded.
- Mediation, including whether a text is a retelling, tourist framing, scholarly framing, catalogue description, entertainment use, or direct public authorship.

## Public Metadata Caution

Public catalogue metadata is not permission to extract restricted cultural knowledge. If a catalogue record indicates restrictions, community control, secret/sacred status, unpublished material, or uncertainty about access, mark it `restricted_excluded` or `uncertain` and do not import full text.

## Automation Boundaries

The first-pass scripts do not scrape aggressively and do not bypass access controls. Collector scripts are templates or stubs. Manual import assumes that the researcher has already verified publicness and scope.
